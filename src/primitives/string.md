# String
```
String: Type
```

Create a string with `'`, `` ` `` or `"`.
All are equivalent.
Declaring strings is similar to string literals in other languages.
```
'hi
';
```
Is fine and is equivalent to 
```
'hi\n';
```

To escape the string, use `\` backslash; for example
```
'Here\'s a string';
```

## Methods


### len
```
len: func () Number;
```
Returns the length of the string
